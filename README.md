# 🍽️ Food Delivery Analysis using Tableau

An interactive Business Intelligence project developed using **Tableau** to analyze food delivery data and generate meaningful business insights. The project includes an interactive dashboard, a Tableau Story with five story points, Tableau Public publishing, Flask-based web integration, and project documentation.

---

# 📖 Project Overview

The **Food Delivery Analysis** project transforms raw food ordering data into interactive visualizations that help analyze customer ordering behaviour, revenue trends, delivery performance, cuisine preferences, and city-wise order distribution.

The project enables users to explore data through interactive filters and Tableau Stories, supporting better business decision-making.

---

# 🎯 Objectives

- Analyze customer food ordering behaviour.
- Identify revenue trends across meal types.
- Evaluate delivery performance.
- Analyze cuisine preferences.
- Compare order distribution across different cities.
- Present business insights through Tableau Dashboard and Story.
- Integrate Tableau Public dashboard with a Flask web application.
